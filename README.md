# Sistema SmartWorking

Implementação das funcionalidades básicas do sistema de Coworking para a disciplina de Engenharia de Software.

## Funcionalidades Implementadas
1. **Login (RF01):** Autenticação de membros.
2. **Visualizar Espaços (RF05):** Listagem de salas e mesas.
3. **Reservar Espaço (RF06):** Criação de reservas com validação de conflito de horário (RN05).

## Como Rodar
1. Certifique-se de ter o Java instalado.
2. Compile os arquivos: `javac *.java`
3. Execute o sistema: `java SmartWorkingSystem`
